# Hito 大聯盟專屬播放器 (Hito MLB Podcast Player)

這是一個專為「Hito 大聯盟」Podcast 聽眾打造的網頁播放器。透過串接官方 RSS Feed，提供更精準的章節跳轉、全域段落檢索以及優化的收聽體驗。

## 系統需求

### 1. 主播放面板 (Upper Panel)
* **最新集數呈現**：首頁自動載入最新一集的 Hito 大聯盟，顯示單集封面、標題、發布日期與總時長。
* **資訊精準度**：
    * **日期顯示**：直接擷取單集描述中的 `Published on:` 字串（例如：`Tue, 24 Feb 2026`），確保與官方原文完全一致，徹底解決時區偏移（跳日）的 Bug。
    * **時長格式**：總時長統一格式化為補足零後的 `HH:MM:SS`（例如：`03:19:10`）。
* **整合導覽**：標題旁提供「原始單集連結」圖示，點擊可直達 RSS.com 網頁。

### 2. 章節互動與單集資訊 (Information & Chapters)
* **章節置頂設計**：章節段落清單位於單集描述上方，方便使用者在收聽前快速瀏覽討論重點。
* **智慧章節解析**：
    * 自動忽略贊助連結（zeczec）前的贅詞，確保第一個章節標題純淨且精確。
    * 自動清理標題中殘留的「本集討論」等前綴關鍵字。
* **即時跳轉播放**：點擊任一章節標題或時間戳（例如：`46:37`），播放器即刻精準跳轉至該段落開始播放。
* **順暢閱讀體驗**：章節下方呈現完整的單集描述（Episode Notes），全頁面移除內置捲軸，改用瀏覽器原生捲動方式。

### 3. 側邊欄集數列表 (Sidebar & Pagination)
* **分頁瀏覽機制**：側邊欄列表以每頁 **10 筆** 顯示歷史集數資訊（包含日期與時長）。
* **雙重分頁控制**：清單上方與下方同步提供 `PREV` 與 `NEXT` 按鈕，提升長列表的操作便利性。
* **一鍵重設功能**：點擊左上角大圖示或分頁列中的封面小圖示，立即清空搜尋狀態並回到第一頁最新集數。

### 4. 全域關鍵字檢索 (Global Search)
* **跨集章節檢索**：支援輸入關鍵字比對所有歷史集數的「段落標題」。
* **便捷搜尋操作**：搜尋框支援按下 `Enter` 鍵立即執行，搜尋結果同樣享有 10 筆一頁的分頁功能。
* **搜尋結果高亮**：在清單中將匹配到的關鍵字以黃色背景（Highlight）標示，點擊後面板自動切換至該集，並直接定位到該段落播放。

## 實作需求

### 1. 後端架構 (Spring Boot)
* **資料來源串接**：指定使用 Hito 大聯盟官方 RSS 網址：`https://media.rss.com/hitomlb/feed.xml`。
* **分層開發設計 (Controller/Service)**：
    * `PodcastController`：處理 API 請求派送與跨域（CORS）設定。
    * `PodcastService`：負責 RSS Feed 抓取（使用 ROME 函式庫）、正規表示式（Regex）解析與業務邏輯處理。
* **強健解析邏輯**：優化 Regex 處理標點符號，確保標題內含「、」或「，」時不會被錯誤切斷。

### 2. 前端實作 (Pure JavaScript + Bootstrap 5)
* **結構分離**：獨立拆分 `index.html`、`style.css` 與 `script.js`，提升程式碼可維護性。
* **穩定綁定機制**：使用 `DOMContentLoaded` 確保 HTML 元素完全載入後才執行事件監聽，杜絕 `null` 指標錯誤。
* **瀏覽器整合**：設定 Chrome 瀏覽器分頁標籤圖示 (Favicon) 為節目封面，方便使用者在多個分頁中快速辨識。
